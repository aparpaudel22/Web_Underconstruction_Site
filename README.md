# Under Construction site

This is a **simple interactive “Under Construction” website** with a **mini game** built using **HTML, CSS, and JavaScript**.
While the main website is under development, visitors can play a small game where they **catch falling blocks** using keyboard or touch controls.

---

##  Live Idea

- Shows an **“Under Construction”** message
- Allows visitors to **play a mini game**
- Saves **high score** in the browser
- Works on **desktop and mobile**

---

## Technologies Used

- **HTML5** – Structure of the page
- **CSS3** – Styling and animations
- **JavaScript (Vanilla)** – Game logic
- **Canvas API** – Drawing the game
- **LocalStorage** – Saving high score
- **Audio** – Background and game-over sounds
